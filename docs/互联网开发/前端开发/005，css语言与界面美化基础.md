# CSS语言与界面美化

CSS（Cascading Style Sheets，层叠样式表）用于控制网页的外观和布局。

## 前置知识：HTML元素关系

### 父元素与子元素

在HTML文档中，元素之间存在嵌套关系：

<div> ← 父元素（parent）
<p> ← 子元素（child）
<span>← 孙子元素
</p>
</div>

/\* 父元素 \*/
div {
padding: 20px;
border: 2px solid #3498db;
}
/\* 子元素 \*/
div > p {
color: red;
}
/\* 孙子元素（所有后代） \*/
div p {
color: blue;
}

### 祖元素与后代元素

祖元素（祖先）包含了子元素和孙子元素，后代元素是被嵌套在里面的所有元素。

/\* 后代选择器：选择div内部所有p元素 \*/
div p { }
/\* 子选择器：只选择div的直接子元素p \*/
div > p { }

### 兄弟元素

兄弟元素是拥有相同父元素的元素：

<ul>
<li>元素1</li> ← li元素2的哥哥
<li>元素2</li> ← li元素1的弟弟
</ul>

* 列表项1（哥哥）
* 列表项2（弟弟）

**提示：**理解元素关系对于使用CSS选择器非常重要，尤其是后代选择器（空格）和子选择器（>）的区别。

## 第一部分：CSS选择器

### 什么是CSS选择器？

选择器用于指定要样式化的HTML元素。

### 基本选择器

| 选择器 | 名称 | 示例 |
| --- | --- | --- |
| 元素选择器 | 选择所有指定元素 | p { } |
| 类选择器 | 选择class属性值相同的元素 | .classname { } |
| ID选择器 | 选择指定id的元素（唯一） | #idname { } |
| 通配符选择器 | 选择所有元素 | \* { } |

### 选择器示例

/\* 元素选择器：选择所有p标签 \*/
p {
color: blue;
}
/\* 类选择器：选择class="red"的元素 \*/
.red {
color: red;
}
/\* ID选择器：选择id="header"的元素 \*/
#header {
background: gray;
}

### 组合选择器

| 选择器 | 名称 | 示例 |
| --- | --- | --- |
| 后代选择器 | 选择元素内部的所有指定元素 | div p { } |
| 子选择器 | 选择元素的直接子元素 | div > p { } |
| 相邻兄弟选择器 | 选择紧邻的元素 | h1 + p { } |
| 群组选择器 | 同时选择多个元素 | h1, h2, h3 { } |

### 伪类和伪元素

/\* 伪类：鼠标悬停时 \*/
button:hover {
background: blue;
}
/\* 伪类：第一个子元素 \*/
li:first-child {
font-weight: bold;
}
/\* 伪元素：首字母 \*/
p::first-letter {
font-size: 2em;
}

这是一个段落，鼠标悬停在下方的按钮上查看效果：

鼠标悬停在我上面

**提示：**伪类用单个冒号（:），伪元素用双冒号（::）。现代CSS推荐使用双冒号。

## 第二部分：CSS属性

### 常用文本属性

p {
color: #333; /\* 文字颜色 \*/
font-size: 16px; /\* 字体大小 \*/
font-family: Arial; /\* 字体类型 \*/
font-weight: bold; /\* 粗体 \*/
text-align: center; /\* 对齐方式 \*/
line-height: 1.8; /\* 行高 \*/

### 常用背景属性

div {

background-image: url(bg.jpg); /\* 背景图片 \*/

background-position: center; /\* 居中 \*/

div {
border: 2px solid red; /\* 简写：宽度 样式 颜色 \*/
border-width: 2px; /\* 边框宽度 \*/
border-style: solid; /\* 边框样式 \*/
border-color: red; /\* 边框颜色 \*/
border-radius: 10px; /\* 圆角 \*/
}

### 常用盒模型属性

盒模型包括：内容(content) + 内边距(padding) + 边框(border) + 外边距(margin)

div {
width: 200px; /\* 宽度 \*/
height: 100px; /\* 高度 \*/
padding: 20px; /\* 内边距 \*/
margin: 10px; /\* 外边距 \*/
box-sizing: border-box; /\* 盒尺寸计算方式 \*/
}

默认盒模型

border-box

**提示：**使用 box-sizing: border-box 可以让padding和border包含在width和height内，计算更方便。

## 第三部分：浮动

### 什么是浮动？

浮动让元素向左或向右移动，其周围的元素也会围绕它排列。

### 浮动示例

.box {
float: left; /\* 左浮动 \*/
margin: 5px;
}
/\* 清除浮动 \*/
.clearfix::after {
content: "";
display: table;
clear: both;
}

### 清除浮动

/\* 方法1：使用clear属性 \*/
.clear {
clear: both;
}
/\* 方法2：使用::after伪元素 \*/
.container::after {
content: "";
display: table;
clear: both;
}

**注意：**浮动会导致元素脱离文档流，父元素高度塌陷，需要使用clearfix技巧或设置父元素overflow: hidden来解决问题。

## 第四部分：定位

### 定位属性

position属性用于控制元素的定位方式。

| 值 | 说明 |
| --- | --- |
| static | 默认值，正常文档流定位 |
| relative | 相对定位，相对于自身原始位置定位 |
| absolute | 绝对定位，相对于最近已定位祖先元素定位 |
| fixed | 固定定位，相对于浏览器窗口定位 |
| sticky | 粘性定位，在滚动范围内是相对定位，超出后固定 |

### 相对定位（relative）

.box {
position: relative;
top: 20px;
left: 20px;
}

### 定位的层级

.box1 {

}
